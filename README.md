# SBW Entity Render Distance

Серверный мод для увеличения дистанции видимости игроков и транспорта.

## Команды (только для операторов)

- `/renderdistance players <64-512>`
- `/renderdistance vehicles <64-512>`

## Конфиг

`config/sbwrender-server.toml`

```toml
playerTrackingRange = 256
vehicleTrackingRange = 256
```
